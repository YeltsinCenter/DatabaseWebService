Сделал тестовое с использованием шаблона ASP.NET Web API, потому что это современный стандарт, хорошо документирован и можно тестировать через Postman. Добавил механизм сессий для сохранения подключения, для быстроты решения храню подключения в sessionId. За обработку http запросов отвечает DatabaseController.cs, а за работу с БД DatabaseService.cs. В I использовал асинк, чтобы интерфейс не зависал. Также добавил очистку старых подключение и простую обработку ошибок. Это версия 1 - v1. В v2 добавил бы ещё и аутентификацию, доработал хранение подключений, добавил логирование и unit-тесты.

Требования:
Visual Studio 2019/2022
SQL Server (любая редакция: Express, Developer, Standard)
.NET Framework 4.7.2

1) Открыть решение
2) В файле Client/Form1.cs в свойство _baseUrl вставить свой порт.
3) Запустить
